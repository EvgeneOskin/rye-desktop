# app

> My superb app


## Dev

```
$ npm install
```

After installing new module with do:
```
$ ./node_modules/.bin/electron-rebuild
```
### Run

```
$ npm start
```

### Build

```
$ npm run build
```

Builds the app for OS X, Linux, and Windows, using [electron-packager](https://github.com/maxogden/electron-packager).
